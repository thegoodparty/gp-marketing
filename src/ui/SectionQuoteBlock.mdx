# Section Quote Block

<Description of={SectionQuoteBlockStories.Default} />

## Overview

The SectionQuoteBlock displays a grid of quote cards with optional header. Each card shows a quote with author attribution (avatar, name, secondary text). Designed for blog quote sections and testimonial-style content.

## Usage

Use this component when:
- Displaying testimonial quotes in a blog or marketing page
- Showing candidate or user success stories in a grid layout
- Presenting multiple quotes with author attribution

## Props

<Controls of={SectionQuoteBlockStories.Default} />

### Prop Details

#### `items` (required)

Array of SectionQuoteCard props. Each item has `copy` (quote text), `author` (name, meta, image).

**Type:** `SectionQuoteCardProps[]`

#### `header` (optional)

HeaderBlock configuration. Use `label: 'Blog Quote'` for the section overline.

**Type:** `HeaderBlockProps`

#### `backgroundColor` (optional)

Background color variant.

**Type:** `'cream' | 'midnight'`

**Default:** `'cream'`

## Grid Layout

- **Mobile**: Single column
- **Desktop**: 2 columns (`md:grid-cols-2`)
- Gap: `gap-6` between cards
