# Arrow Right Icon

<Description of={ArrowRightIconStories.Default} />

## Overview

The ArrowRightIcon component is a reusable SVG icon component that displays a right-pointing arrow. It's used throughout the application to indicate navigation, links, or forward movement. The component accepts customizable size and styling props.

## Usage

Use this component when:
- Displaying navigation indicators in lists or cards
- Showing clickable links or buttons
- Indicating forward movement or next actions
- Creating consistent arrow icons across the application

### Prop Details

#### `size` (optional)

The size of the arrow icon in pixels. Both width and height will be set to this value.

**Type:** `number`

**Default:** `32`

**Example:** `24`, `32`, `48`

#### `className` (optional)

Additional CSS classes to apply to the SVG element. Useful for styling, colors, or animations.

**Type:** `string`

**Example:** `"text-blue-500"`, `"opacity-50"`

#### SVG Props

The component accepts all standard SVG element props via spread props, allowing for full customization of the SVG element's attributes.

**Type:** `SVGProps<SVGSVGElement>`

## Common Use Cases

### Basic Usage

```tsx
<ArrowRightIcon />
```

### Custom Size

```tsx
<ArrowRightIcon size={24} />
```

### With Styling

```tsx
<ArrowRightIcon size={32} className="text-blue-500" />
```

### In Links

```tsx
<Anchor href="/page">
  <span>View more</span>
  <ArrowRightIcon size={24} className="ml-2" />
</Anchor>
```

## Styling

The icon uses `currentColor` for the fill, which means it will inherit the text color from its parent element. This allows for easy color customization via CSS classes.

## Accessibility

- The icon is decorative and should be used alongside text labels
- When used in links, ensure the link text provides context
- For purely decorative use, consider adding `aria-hidden="true"`
