# Candidates Card 2

<Description of={CandidatesCard2Stories.Default} />

## Overview

CandidatesCard2 is a specialized candidate card component for GoodParty-empowered candidates. It displays candidate information with a GoodParty logo badge overlay on the avatar, includes "Empowered by goodparty.org" text, and uses a bright yellow border to distinguish it from standard candidate cards.

## Usage

Use this component when:
- Displaying GoodParty-empowered candidates in a CandidatesBlock grid
- Highlighting candidates who are part of the GoodParty program
- Creating visual distinction between standard and GoodParty candidates

### Prop Details

#### `name` (required)

The candidate's full name.

**Type:** `string`

#### `partyAffiliation` (required)

The candidate's party affiliation or role title.

**Type:** `string`

#### `href` (required)

The URL path to the candidate's profile page.

**Type:** `string`

#### `avatar` (optional)

The candidate's profile image. Can be a SanityImage object or a string URL.

**Type:** `SanityImage | string`

#### `isGoodPartyCandidate` (optional)

Flag indicating this is a GoodParty candidate. Used for type checking and component selection in CandidatesBlock.

**Type:** `boolean`

#### `className` (optional)

Additional CSS classes to apply to the component.

**Type:** `string`

## Visual Distinctions

CandidatesCard2 differs from CandidatesCard1 in several ways:

1. **Border Color**: Uses `border-bright-yellow-600` instead of `border-black/8`
2. **Hover Effect**: Background changes to `hover:bg-bright-yellow-50` on hover
3. **Badge Overlay**: GoodParty logo badge positioned at bottom-right of avatar
4. **Attribution Text**: Includes "Empowered by goodparty.org" text below party affiliation

## Avatar Badge

The GoodParty logo badge is positioned absolutely at the bottom-right of the avatar:
- Size: 50px × 35px
- Contains the GoodParty logo SVG
- Overlays the avatar image

## Avatar Fallback

When no `avatar` is provided, the component generates initials from the candidate's name (same as CandidatesCard1):
- Takes the first letter of each word
- Converts to uppercase
- Displays up to 2 characters
- Badge still appears on initials avatar

## Layout

Same responsive flex layout as CandidatesCard1:
- Mobile: Vertical stack
- Desktop: Horizontal layout with avatar, content, and link

## Accessibility

- Uses semantic `<article>` element
- Avatar images include proper alt text
- "View profile" link is keyboard accessible
- Attribution text provides context about GoodParty empowerment
